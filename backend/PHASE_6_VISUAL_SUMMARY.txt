│ TRACE ID PROPAGATION FLOW                                                       │
└─────────────────────────────────────────────────────────────────────────────────┘

    HTTP Request
         │
         ├─→ traceIdMiddleware
         │   └─→ Generate/Extract trace ID
         │       └─→ Attach to req.traceId
         │           └─→ Set X-Trace-ID header
         │
         ├─→ Route Handler
         │   └─→ Use req.traceId
         │       └─→ Emit Event with traceId
         │
         ├─→ Event Emission
         │   ├─→ Local Event Bus
         │   │   └─→ Listener executes with traceId
         │   │       └─→ Structured logger records
         │   │
         │   └─→ Redis Event Bus
         │       └─→ Distributed listener with traceId
         │           └─→ Structured logger records
         │
         ├─→ Job Queuing
         │   └─→ Queue job with traceId
         │       └─→ Structured logger records
         │
         ├─→ Worker Processing
         │   └─→ Process job with traceId
         │       └─→ Emit events with traceId
         │           └─→ Structured logger records
         │
         └─→ Response
             └─→ Send with X-Trace-ID header
